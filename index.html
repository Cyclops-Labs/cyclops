User Data Records Micro Service

UDR Service connects to cloud installation such as OpenStack, Cloudstack, etc and collects the data. The data is harmonized and saved into a Time Series Based database. The harmonized usage data is exposed through REST APIs.

Salient Features

  • Data collection through connectors
  • Usage data transformation
  • Availability of usage data via REST APIs
  • External SaaS/PaaS usage data insertion via REST APIs

Rating & Charging Micro Service

Dynamic rates are generated through a rule engine by considering the various policies, environmental factors, etc. A charge for the usage made is calculated by using the harmonized data from UDR Service.

Salient Features

  • Dynamic Rating & Charging
  • Rule engine based Rating engine
  • CDR persistence for data redundancy
  • Exposing CDR through REST APIs

Billing Micro Service

Various plans, discounts and coupons are considered through a billing engine to calculate the price to be paid by the user. Once the invoice is generated, the service connects to the payment gateway for payment fulfilment

Salient Features

  • Rule engine based Billing engine
  • Connection to payment gateway
  • Exposing billed data through REST APIs
